Today I would like to announce the first release of my first program, Drew's Application! It does not sound like much but sometime I will make the title better. Here is a screenshot:


This application is currently just an "About" screen. In future versions there will be more things.
Here are the requirements:

  • Windows XP SP3 or higher
  • 32-bit or 64-bit
  • Microsoft .NET Framework 4 (Google it. First result.)
  • Processor: Pentium 1 GHz or higher with 512 MB of RAM or more
  • Minimum disk space: (Needed for the .Net Framework)
    • x86 (32-Bit) – 850 MB
    • x64 (64-Bit) – 2 GB
  • Prerequisites: (For the .Net Framework 4)
    • Windows Installer 3.1 or later
    • Internet Explorer 5.01 or later
