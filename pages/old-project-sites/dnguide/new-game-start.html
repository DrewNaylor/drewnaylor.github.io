How to start a new game.

If you have ever wondered how to start a new game, here is some information:

Step 1:
Set up the Wii according to the Wii Operations Manuel (sic).
Put the CD (sic) into the drive on the front of the Wii. It should start up.
If the Wii Menu needs upgrading, follow the on-screen instructions.



Step 2:
On the Main Menu, press A+B, (A is on the front; B is on the back).
Then, click on one of the files. A menu will ask you what icon you want to use.
When you are finished setting up, click on the account to get started.
