  • Project64

    Nintendo 64 emulator for Windows

    Note: Some versions of Project64, specifically at least version 2.0 (but version 1.6 should be safe and it's what I personally recommend) contain adware in the installer. Only use version 1.6 if you don't want to be bothered with the adware. If you know what you're doing, you can pass the installer the "/noads" flag and the version 2.2 installer shouldn't give you ads. I have not looked into newer versions, so I don't know how their installers are.
