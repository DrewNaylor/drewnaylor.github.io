This is a test page, mainly for the Mario mouse JS thing.

Here's how I did it (the right way) :

  1. Made a file called MarioFollowsCursorScript.js in the js directory in the root of my website.
  2. Placed the JavaScript code from here into the MarioFollowsCursorScript.js file, with modifications to use Mario sprites and help from my friend, Dustin Herl, (for the original modifications,) and removed the script tags.
  3. Made a new file called ShowMarioCursor.html to house the HTML from the modified Rainbow Arch code and placed it in the _includes directory. At the bottom of the document I placed a script src tag that references the MarioFollowsCursorScript.js from earlier.
  4. Put an include ShowMarioCursor.html tag (surrounded by % and {}) at the top of this page, but below the Jekyll page markup stuff with the 3 dashes.
